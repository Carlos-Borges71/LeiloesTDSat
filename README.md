Projeto LeiloesDTSat

Projeto Java para gerenciamento de produtos (CRUD) usando Swing/NetBeans e conexão com banco de dados .


Tecnologias
- Java SE
- Swing
- DAO/DTO
- NetBeans
- Banco de Dados MSQL

Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Alteração
- Exclusão
- Conexão com banco e operações CRUD
